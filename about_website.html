Website Project

This project was assigned to our team as the third homework in the Principles of Information Systems course. This task consisted of two stages: individual and team. The individual part consisted of taking courses at the Code Academy , and the team part was using the acquired knowledge to create the site. The languages you need to build a website are HTML , CSS , JavaScript and they can be learned on many platforms, but our team recommends the Code Academy. This environment will help you to understand the fundamental concepts of these languages and give you only the knowledge you need. The teamwork part of the task took place on the GitHub platform. GitHub is the largest web service for hosting IT projects and their joint development.
